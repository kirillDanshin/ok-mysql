# Ok-MySQL

MySQL query time logger

# Requirements to run
Needs to run under root user.

# Compiling dev branch
### Release
```
git clone git@github.com:kirillDanshin/ok-mysql.git
go get
make rel
```

### Debug
```
git clone git@github.com:kirillDanshin/ok-mysql.git
go get
make debug
```
